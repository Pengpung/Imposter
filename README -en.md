# Welcome to "Who's the Spy"

This is a little game. Currently, the dictionary only contains countries, but in version 2.0, I added an "Import Dictionary" function, allowing you to add your own words. However, for the most stable experience, it's best not to add words (as the author hasn't had time to test it and doesn't know if it will work correctly).

Main branch only available for the web version (powered by Lovable): impostely.lovable.app
